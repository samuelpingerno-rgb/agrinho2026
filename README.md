🌱💵 Sobre o Projeto 📉
Compre os produtos certificados. Fortaleça o Agronegócio Brasileiro!
O objetivo deste Web Site é ajudar a identificar selos de Produção Sustentável em produtos do AGRO (Agronegócio) brasileiro. Criamos um Guia ON-LINE dos produtos com o selo de produção sustentável. É uma página dedicada ao consumidor, na qual, o mesmo pode verificar se o produto é resultado da produção sustentável. A nossa sugestão é de que em cada embalagem dos produtos AGRO, deveria conter um QR CODE que direciona o consumidor para o site e o mesmo poderá realizar realizar a consulta dos produtos certificados.
Instruções da pesquisa:
Você pode realizar a busca pelo nome do “selo ou Certificação” e também pelo nome do Produto. Exemplo: “café" – é o produto “Rainforest Alliance" – é o selo/certificação
